# 应用中心 (Apps Center)

## 概述

重新设计的应用中心是一个功能丰富的应用集成管理平台，提供现代化的用户界面和强大的筛选功能，帮助用户发现、管理和连接各种第三方应用。

## 主要特性

### 🎯 智能筛选系统
- **分类筛选**: 支持按通讯、生产力、开发、设计、金融、娱乐、工具、安全等分类浏览
- **价格筛选**: 按免费、免费增值、付费、订阅等价格模式筛选
- **评分筛选**: 根据用户评分范围筛选应用
- **状态筛选**: 按连接状态、精选应用等条件筛选
- **搜索功能**: 支持应用名称、描述、开发者、标签等多维度搜索

### 📊 丰富的数据展示
- **应用统计**: 显示总应用数、已连接数、精选应用数、平均评分等关键指标
- **详细应用信息**: 包含应用图标、名称、开发者、描述、版本、更新日期等
- **用户评价**: 显示应用评分和评价数量
- **价格信息**: 清晰展示应用的价格模式和费用

### 🎨 现代化的界面设计
- **双视图模式**: 支持网格视图和列表视图切换
- **响应式设计**: 适配不同屏幕尺寸
- **悬停效果**: 卡片悬停时的平滑动画和阴影效果
- **状态指示**: 通过颜色标识应用连接状态和运行状态

### 🔍 高级功能
- **应用详情模态框**: 点击查看应用的完整信息，包括功能权限、系统要求等
- **连接管理**: 一键连接/断开应用集成
- **排序功能**: 支持按名称、评分、评价数、更新时间排序
- **快速筛选**: 显示当前激活的筛选条件，支持快速清除

## 应用数据结构

每个应用包含以下信息：
- 基本信息: ID、名称、图标、描述、分类
- 状态信息: 连接状态、运行状态、精选标识
- 评价信息: 评分、评价数量
- 版本信息: 版本号、最后更新日期、开发者
- 价格信息: 价格模式、费用、周期
- 权限信息: 应用需要的权限列表
- 标签信息: 应用的功能标签
- 系统要求: 最低版本、存储空间、内存要求

## 技术实现

### 组件架构
- `AppCard`: 应用卡片组件，显示应用基本信息
- `AppDetail`: 应用详情模态框组件
- `AppFilters`: 筛选器组件，提供各种筛选功能
- `useApps`: 组合式函数，管理应用数据和筛选逻辑

### 数据管理
- 使用 Vue 3 的组合式 API 进行状态管理
- 响应式筛选和排序功能
- 本地数据存储，支持未来扩展为 API 调用

### 样式设计
- 使用 Tailwind CSS 进行样式设计
- 现代化的卡片设计和布局
- 响应式网格系统
- 平滑的过渡动画效果

## 使用说明

1. **浏览应用**: 在网格或列表视图中浏览所有可用应用
2. **筛选应用**: 使用左侧筛选器按分类、价格、评分等条件筛选
3. **搜索应用**: 在搜索框中输入关键词快速找到应用
4. **查看详情**: 点击应用卡片或详情按钮查看完整信息
5. **管理连接**: 点击连接按钮管理应用集成状态
6. **排序应用**: 使用排序下拉菜单调整应用显示顺序

## 扩展性

应用中心设计具有良好的扩展性：
- 支持添加新的应用分类
- 可扩展筛选条件
- 支持应用截图展示
- 预留 API 集成接口
- 支持应用安装和配置流程

这个新的应用中心为用户提供了更加直观、高效的应用发现和管理体验，是现代应用集成平台的理想解决方案。
